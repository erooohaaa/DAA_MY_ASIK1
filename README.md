# Divide and Conquer Algorithms Project

## Overview
Implementation and analysis of classic divide-and-conquer algorithms:
- MergeSort
- QuickSort
- Deterministic Select (Median-of-Medians)
- Closest Pair of Points

## Tech Stack
- Java 17
- Maven
- JUnit 5 for testing
